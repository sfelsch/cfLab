# CF Lab Website

Sitio web del laboratorio **CF Lab**, publicado con **GitHub Pages**.

## 🚀 Demo online
👉 https://sfelsch.github.io/cfLab/

## 📂 Estructura
- `index.html` → Página principal
- `assets/css/` → Estilos (`main.css`, `custom.css`)
- `assets/js/` → Scripts
- `assets/img/` → Imágenes del sitio
- `assets/vendor/` → Librerías de terceros
- `forms/` → Archivos de formularios (a configurar)

## 🛠️ Stack
- HTML5 · CSS3 · JavaScript
- Bootstrap 5
- Bootstrap Icons, AOS, Glightbox, Swiper, PureCounter

## 🌐 Deploy (GitHub Pages)
Este repo publica automáticamente con **GitHub Pages** desde la rama **main**.  
Cada `git push` a `main` dispara el redeploy.

## 📝 Edición rápida
- **Colores/branding:** `assets/css/custom.css`
- **Hero / About / Services:** editar secciones en `index.html`
- **Imágenes:** reemplazar en `assets/img/` (mantener nombres o actualizar rutas)
